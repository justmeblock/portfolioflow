# Personal Portfolio

A modern, responsive personal portfolio website built with Vite + React.

## Features

- Smooth animations
- Responsive design
- Sections for About, Projects, Skills, Experience, and Contact
- Tailwind CSS styling
- Framer Motion for animations

## Development

```bash
npm install
npm run dev
```

Runs on `http://localhost:8080`

## Build

```bash
npm run build
```

## Customize

Replace placeholder text, images, and links with your personal information.
